## 1. Definición y características

- Git es un sistema de control de versiones distribuido.
- Trata los proyectos software como repositorios de archivos.
- Cada usuario tiene una copia local con sus propias diferencias.
- Permite también tener un repositorio central por proyecto.
- Se utiliza por consola de comandos pero hay clientes gráficos.
- Fue creado por Linus Torvalds, creador de Linux.

## 2. Instalación

- Git: [https://git-scm.com/downloads](https://git-scm.com/downloads)
- SourceTree: [https://www.sourcetreeapp.com/](https://www.sourcetreeapp.com/)

## 3. Configuración

Después de instalarlo, se recomienda dar valor a estos parámetros, ya que de lo contrario el cliente por consola suele emitir un error (en Windows especialmente):

    git config --global user.name "Mi Nombre Completo"
    git config --global user.email "miemail@dominio.com"

Estos comandos deben realizarse a través de la consola, en el caso de Windows sería CMD, PowerShell, Git Bash o Git CMD.
